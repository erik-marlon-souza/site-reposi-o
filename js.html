O que é JS

JavaScript é uma linguagem de programação voltada para aplicações web, de fácil execução e que pode ser rodada a partir de um navegador de internet. Hoje, é um dos mais populares do mundo, o que o torna essencial para programadores. O mercado está cada vez mais se voltando para a web, o que mostra como o JavaScript está ganhando uma posição de liderança.

O impacto que JavaScript tem atualmente na tecnologia e na forma como empresas e usuários comuns lidam com recursos digitais

Por que estudar JS?

  1. O JavaScript umas das linguagens de programação mais populares e usadas por desenvolvedores em todo o mundo
  2. Possui uma grande quantidade de conteúdos disponíveis na Web
  3. É de fácil aprendizagem
  4. E é muito versátil
